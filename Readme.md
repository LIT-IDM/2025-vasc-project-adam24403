Christmas Tree AR App

An AR app perfect for Christmas!.

This project allows users to view an animated 3D Christmas Tree in Augmented Reality by scanning a printed AR marker.
The app is built in Unity using AR Foundation and is compatible with Android devices.

Download the App

Download ChristmasTree.apk
(Hosted on GitHub Pages)

Once downloaded, follow the installation steps below.

How to Install on Android

Tap Download ChristmasTree.apk

Your device may warn about unknown apps — tap Install anyway.

Once installed, open the app from your home screen.

Grant camera permission if asked.

How to Use the app

Open the Christmas Tree AR App.

Point your camera at the AR Marker.

A Christmas Tree will appear on the marker in AR.
